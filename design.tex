\section{Rive日本語入力system}

Rive日本語入力システムは以下のものによって構成されている

\begin{itemize}
  \item Rive Client
  \item Rive Server
  \item Rive Analytics
  \item Rive BtachProcessing
  \item Rive Webservice
\end{itemize}

これら５つのシステムがお互いに作用することで
Rive日本語入力システムが実現されている。
